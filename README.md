### Install

Clone project using Git:

```
git clone https://github.com/babus6332/Big_Data_project.git
```

Go to created directory:

```
cd Big_Data_project
```

Install jupyter using pip:

```
pip install jupyter
```

Run jupyter:

```
jupyter notebook
```
